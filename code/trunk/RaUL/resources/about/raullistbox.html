RaUL:Listbox

The Listbox Widget allows the user to make one or multiple selections from a set of choices. Special properties include list, multiple and row. In case of the row and multiple properties the rendering in HTML is straightforward. In both cases it is rendered as a select input box displayed as a multirow selection box (row) and with the ability to select more than one value (multiple). If the list property is used, the Listbox needs to reference a collection of Listitems. Listitems are required to define a value and label property and can be defined as checked. The listing below defines a "Price Currency Option" Listbox from our ProductAdd example. Any Listbox must bind to an RDF collection with the same number of node elements as there are Listitems. Similar to the data binding in the Textbox we use RDF reification to associate the value for the Listbox to a predicate in an ontology, in the example below, the GoodRelations hasCurrency property. Additionally to filling in the value of the selected Listitem to the reified rdf:object triple, the a checked property is added to the any Listitem that was selected by the user.

<rdf:RDF
  xmlns:rdf="http://www.w3.org/1999/02/22-rdf-syntax-ns#"	
  xmlns:raul="http://purl.org/NET/raul#"
  xmlns:gr="http://purl.org/goodrelations/v1#">

<rdf:Description rdf:about="http://w3c.org.au/raul/service/public/forms/productAdd#price_currency_options">
  <rdf:type rdf:resource="http://purl.org/NET/raul#Listbox"></rdf:type>
  <raul:id>price_currency_options</raul:id>
  <raul:name>price_currency_options</raul:name>
  <raul:list>http://w3c.org.au/raul/service/public/forms/productAdd#list_price_currency_options</raul:list>
  <raul:value>http://w3c.org.au/raul/service/public/forms/productAdd#value_price_currency_options</raul:value>
</rdf:Description>

<rdf:Description rdf:about="http://w3c.org.au/raul/service/public/forms/productAdd#value_price_currency_options">
  <rdf:subject></rdf:subject>
  <rdf:predicate>gr:hasCurrency</rdf:predicate>
  <rdf:object></rdf:object>
</rdf:Description>

<rdf:Description rdf:about="http://w3c.org.au/raul/service/public/forms/productAdd#list_price_currency_options">
  <rdf:type rdf:resource="http://www.w3.org/1999/02/22-rdf-syntax-ns#Seq"></rdf:type>
  <rdf:_1 rdf:resource="http://w3c.org.au/raul/service/public/forms/productAdd#price_currency_options_0"/>
  <rdf:_2 rdf:resource="http://w3c.org.au/raul/service/public/forms/productAdd#price_currency_options_1"/>
  ...
</rdf:Description>

<rdf:Description rdf:about="http://w3c.org.au/raul/service/public/forms/productAdd#price_currency_options_0">
  <rdf:type rdf:resource="http://purl.org/NET/raul#Listitem"/>
  <raul:label>USD</raul:label>
  <raul:value>USD</raul:value>
</rdf:Description>

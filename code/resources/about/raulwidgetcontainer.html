RaUL:WidgetContainer

A WidgetContainer groups Widgets (Textboxes, Listboxes, Buttons etc.) together. It is similar to a form container in XHTML. In fact, the ActiveRaUL processor renders every WidgetContainer as a form in XHTML. A WidgetContainer defines, among other things, a method property to define the type of form submission and to which endpoint in the ActiveRaUL Web service it is mapped to. The ordering of the form controls in a WidgetContainer is defined with an RDF collection. When rendering the XHTML+RDFa code, the ActiveRaUL processor determines the positioning of the control elements based on this RDF collection. For example, in the productAdd form below, the product Widget is rendered first, followed by the update Widget etc.

<rdf:RDF	
  xmlns:rdf="http://www.w3.org/1999/02/22-rdf-syntax-ns#"	
  xmlns:raul="http://purl.org/NET/raul#">

<rdf:Description rdf:about="http://w3c.org.au/raul/service/public/forms/productAdd#container">
  <rdf:type rdf:resource="http://purl.org/NET/raul#WidgetContainer"></rdf:type>
  <raul:method>http://w3c.org.au/raul/service/public/forms/productAdd#method</raul:method>
  <raul:id>productAdd>/raul:id>
  <raul:list>http://w3c.org.au/raul/service/public/forms/productAdd#containerList</raul:list>
</rdf:Description>

<rdf:Description rdf:about="http://w3c.org.au/raul/service/public/forms/productAdd#method">
  <rdf:type rdf:resource="http://purl.org/NET/raul#CREATEOperation"></rdf:type>
</rdf:Description>

<rdf:Description rdf:about="http://w3c.org.au/raul/service/public/forms/productAdd#containerList">
  <rdf:type rdf:resource="http://www.w3.org/1999/02/22-rdf-syntax-ns#Seq"></rdf:type>
  <rdf:_1 rdf:resource="http://w3c.org.au/raul/service/public/forms/productAdd#product"></rdf:_1>
  <rdf:_2 rdf:resource="http://w3c.org.au/raul/service/public/forms/productAdd#update"></rdf:_2>
  ...	
</rdf:Description>

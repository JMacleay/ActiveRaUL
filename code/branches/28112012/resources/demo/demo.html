FOAF Editor

Our second demo is motivated by the most popular RDF files on the Semantic Web (Friend of a Friend (FOAF)). FOAF is used to describe persons and weak relationships between persons (e.g., a person knows another person). Many ways exist to create a FOAF file. Most people create them manually, but there are also domain-specific tools, like FOAF-a-matic that can generate them. In our demo, we replicate the functionality provided by FOAF-a-matic. Instead of developing a specific Web page for a specific ontology (i.e. FOAF), we created a RaUL RDF model to create, retrieve and update the RDF graph describing a Person.
